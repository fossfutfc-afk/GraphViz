# GraphViz 使用说明

> 适用版本：v1.2.0+

GraphViz 是一款基于 C++/Qt6 的图论可视化桌面工具。支持有向图和无向图的文本输入、力导向自动布局、鼠标拖拽交互，以及 10 种经典图论算法的高亮展示。

---

## GUI 布局

```
┌──────────────────────────────────────────┐
│ 菜单栏: 文件 | 编辑 | 视图 | 帮助        │
├──────────────────┬───────────────────────┤
│ 左侧面板 (编辑器) │ 右侧画布 (图可视化)   │
│                  │                       │
│ 输入图数据文本    │ 力导向布局渲染        │
│ 每行一条边/顶点  │ 鼠标拖拽调整位置      │
│                  │                       │
├──────────────────┴───────────────────────┤
│ 底部控制栏: 算法选择 | 参数 | 按钮       │
├──────────────────────────────────────────┤
│ 状态栏: 解析结果 / 算法输出 / 更新提示   │
└──────────────────────────────────────────┘
```

## 输入格式参考

每行一条边或一个孤立点，支持 `#` 行注释和空行。顶点名可包含字母、数字、中文、`-`；含空格或引号时用 `"..."` 包裹，内部 `\"` 表示转义引号。仅含顶点名（无操作符）的行创建孤立点。

### 基本格式

| 格式 | 含义 |
|------|------|
| `A---B` | 无向边，权重 1.0 |
| `A-3--B` | 无向边，权重 3 |
| `A-->B` | 有向边 A→B，权重 1.0 |
| `A-2.5->B` | 有向边 A→B，权重 2.5 |

### 扩展格式

| 格式 | 含义 |
|------|------|
| `A --- B` | 空格分隔 |
| `A-----B` | 多余 `-` |
| `A-2-B` | 单 `-` 包裹权重，无向 |
| `A<-->B` | 双向箭头，无向 |
| `A<---B` | 左向箭头，有向 B→A |
| `A<-3--B` | 左向带权，有向 B→A |
| `"A B"---"C D"` | 含空格顶点名 |
| `"\"X\""---"\"Y\""` | 顶点名为 `"X"` 和 `"Y"` |
| `A-"-2"->B` | 负权重 -2 |
| `A---A` | 无向自环 |
| `A-->A` | 有向自环 |

### 同名节点

| 格式 | 含义 |
|------|------|
| `2(1)---5` | 顶点 "2"，内部标识 `2#1`，画布显示 `2` |
| `2(2)---3` | 另一个顶点 "2"，内部标识 `2#2`，画布显示 `2` |

- `name(N)` — N 为非负整数，括号紧贴顶点名末尾
- 不含后缀时行为不变（向后兼容）
- 引号名不触发后缀解析

### 孤立点

| 格式 | 含义 |
|------|------|
| `A` | 无引号孤立点（无空格、无 `-`） |
| `"孤立节点"` | 引号包裹孤立点（支持任意内容） |
| `"A-B"` | 含 `-` 的顶点名，引号避免歧义 |

- 一行仅含顶点名无操作符时，创建无边孤立点
- 保存时同步输出孤立点，加载不丢失

### 错误拒绝

| 格式 | 原因 |
|------|------|
| `A>-<B` | `>` 不在末尾 |
| `A-<-B` | `<` 不在开头 |
| `A>--B` | 以 `>` 开头 |

## 图论算法（10 种）

在底部控制栏选择算法，可选填起点/终点参数，点击"执行"。

| # | 算法 | 方法 | 参数 | 高亮色 |
|---|------|------|------|--------|
| 1 | 最短路径 | Dijkstra | 起点 + 终点 | 绿色 |
| 2 | 最小生成树 | Kruskal | — | 橙色 |
| 3 | 关节点 & 桥 | Tarjan DFS | — | 红节点 / 紫边 |
| 4 | 欧拉回路 | Hierholzer + 回溯 | 起点（可选） | 蓝色 |
| 5 | 欧拉通路 | Hierholzer + 回溯 | 起点（可选） | 蓝色 |
| 6 | 哈密顿回路 | 回溯 + 剪枝 (≤20顶点) | 起点（可选） | 青色 |
| 7 | 哈密顿通路 | 回溯 + 剪枝 (≤20顶点) | 起点（可选） | 青色 |
| 8 | 连通分量 | BFS | — | 多色 |
| 9 | 强连通分量 | Kosaraju | — | 多色 |
| 10 | 平面性检测 | Euler公式 + K5/K3,3 | — | 状态栏 |

### 算法说明

**最短路径 (Dijkstra)**
- 支持有向/无向混合图，权重为 double
- 须在"起点"和"终点"输入框填写顶点名

**最小生成树 (Kruskal)**
- 仅作用于无向边，不连通图返回提示

**关节点 & 桥 (Tarjan)**
- 关节点（割点）红色高亮，桥（割边）紫色高亮

**欧拉回路 / 通路 (Hierholzer)**
- 小图（≤15顶点, ≤30边）收集多条解，大图贪心单解
- 支持指定起点（"起点"输入框）
- 回路须所有顶点度数为偶；通路恰有 0 或 2 个奇度顶点

**哈密顿回路 / 通路 (回溯)**
- 顶点数 ≤ 20，超过自动跳过
- 收集所有解（上限 100），用「上一解 / 下一解」按钮切换

**平面性检测**
- 使用 Euler 公式、K5/K3,3 子图判定

**连通分量 (BFS) / 强连通分量 (Kosaraju)**
- 各分量独立着色，大小降序排列

## 文件操作

| 操作 | 说明 |
|------|------|
| 打开 (`Ctrl+O`) | 加载 `.graph` 或其他文本文件 |
| 保存 (`Ctrl+S`) | 将当前编辑器内容保存为文件 |

`samples/` 目录提供了多个示例图：
- `planar-cube.graph` — 平面图（正方体）
- `nonplanar-k5.graph` — 非平面图（K5 完全图）
- `euler-circuit.graph` — 欧拉回路示例
- `hamilton-multi.graph` — 多解哈密顿回路
- `shortest-path.graph` — 最短路径示例
- `showcase.graph` — 综合展示（全特性）

## 渲染特性

- **箭头**：有向边终点绘制三角形箭头
- **自环**：弧线形式渲染在顶点上方
- **平行边**：自动垂直偏移 ±7px 分开
- **权重显示**：图中任一边显式提供权重时，全部边显示权值；否则全部隐藏
- **高亮**：每种算法有专属颜色（绿/橙/红紫/蓝/青/多色）
